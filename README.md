# mqttClientsDemo  

## 功能介绍  
MQTT多客户端模拟工具  
支持批量连接MQTT Broker  

## V1 
支持真实设备更新包下载  

## V2  
1、连接、订阅、数据交互为连续触发，即不等待所有设备连接完成才执行订阅，不等待所有设备订阅完成再执行数据交互  
2、定时上报数据，则仅会对数据交互完成后的设备进行数据上报，仅对目前上线完成的  
3、数据展示，异步进行
